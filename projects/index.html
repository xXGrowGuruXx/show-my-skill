Zauberer & Krieger - Round based Game

Spielprinzip →
- Es findet in er Windows Console statt
- Es gibt 2 Charaktere
- Jeder Charakter hat andere Aktionen und Werte
- Beim Start kann man seinen Charakter wählen
- Gegner wählt automatisch den anderen

Krieger →
- standart Schaden: 6
- Schaden an verwundbaren Gegner: 10
- kann ausweichen für 3 Runden

Zauberer →
- standart Schaden: 3
- Effektschaden: 12
- kann einen Zaubertrank brauen. Der Effekt hält für 5 Runden aber macht dich auch verwundbar.

Sprachen: C#
Tools: Windows Console
